# NEOBUSHI
**My first game project: a 2D mech-themed arena roguelite in Godot 3.5.3.**

## About the Project
NEOBUSHI is a fast-paced arena shooter where you control a mech fighting off waves of enemies. This project serves as my introduction to game development and version control.

## Development
This game is being developed with the assistance of **Gemini**, utilizing AI-collaboration for coding mechanics, troubleshooting, and project structure.

## How to Play
* Currently in active development.
* Built using the Godot Engine.
